# React + TypeScript + Ant Design + Tailwind CSS Starter

Starter template with TypeScript, Ant Design, and Tailwind CSS pre-configured.

## Tech Stack

- **React 19** - UI library
- **TypeScript** - Type safety
- **Ant Design 5** - UI component library
- **Tailwind CSS 4** - Utility-first CSS framework
- **Vite 7** - Fast build tool
